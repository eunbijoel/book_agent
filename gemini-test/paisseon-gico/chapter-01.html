1장. 파이썬 시작하기: 환경 설정과 첫 코드

프로그래밍, 그 이름만 들어도 왠지 모르게 어렵고 복잡하게 느껴지시나요? 컴퓨터 화면 가득한 알 수 없는 기호들, 수학 공식처럼 보이는 코드들 때문에 시작조차 망설이셨을지도 모릅니다. 하지만 걱정하지 마세요. 마치 외국어를 배우듯, 가장 쉬운 단어와 문장부터 시작하면 됩니다. 그리고 그 시작을 위한 최고의 언어가 바로 파이썬(Python)입니다.

이번 장에서는 여러분이 프로그래밍의 세계로 첫발을 내딛는 데 필요한 모든 준비를 마칠 것입니다. 파이썬이 무엇인지 이해하고, 컴퓨터에 파이썬 개발 환경을 구축하며, 마침내 여러분의 손으로 직접 작성한 첫 번째 코드를 실행하는 짜릿한 경험을 하게 될 것입니다. 이 여정은 결코 어렵지 않습니다. 친근하고 명확한 안내를 따라가다 보면 어느새 여러분도 코드를 이해하고 작성하는 즐거움을 느끼게 될 것입니다. 자, 이제 파이썬과 함께 새로운 세상을 탐험할 준비가 되셨나요?

1. 파이썬, 당신은 누구인가요?

1980년대 후반, 귀도 반 로섬(Guido van Rossum)이라는 네덜란드 개발자는 '모두를 위한 프로그래밍 언어'라는 철학 아래 파이썬을 만들었습니다. 당시의 다른 프로그래밍 언어들이 복잡한 문법과 어려운 사용법으로 개발자들의 진입 장벽을 높였던 것에 반해, 귀도는 누구나 쉽고 재미있게 코딩할 수 있는 언어를 꿈꿨습니다. 그의 바람대로 파이썬은 간결하고 읽기 쉬운 문법을 자랑하며, 오늘날 전 세계에서 가장 사랑받는 프로그래밍 언어 중 하나로 자리매김했습니다.

그렇다면 파이썬은 정확히 어떤 언어일까요? 파이썬은 고수준(High-level), 인터프리터(Interpreted) 방식의 범용 프로그래밍 언어입니다. 이 용어들이 생소하게 들릴 수 있으니 하나씩 살펴보겠습니다.

  • 고수준 언어 (High-level Language): 컴퓨터가 이해하는 언어(기계어)와는 달리, 사람이 이해하기 쉬운 문법으로 작성된 언어를 말합니다. 파이썬은 영어와 비슷한 문법 구조를 가지고 있어, 마치 영어를 읽듯이 코드를 이해할 수 있습니다.
  • 인터프리터 언어 (Interpreted Language): 작성된 코드를 한 번에 기계어로 번역하여 실행하는 대신, 한 줄씩 읽어 즉시 실행하는 방식을 사용합니다. 덕분에 코드를 작성하고 바로 결과를 확인할 수 있어 개발 속도가 빠르다는 장점이 있습니다.
  • 범용 프로그래밍 언어 (General-purpose Language): 특정 용도에 국한되지 않고 다양한 분야에서 활용될 수 있는 언어입니다. 파이썬은 웹 개발, 데이터 분석, 인공지능, 자동화, 게임 개발 등 거의 모든 분야에서 활용됩니다.

이처럼 파이썬은 배우기 쉽고, 활용 범위가 넓으며, 코드를 작성하는 재미까지 선사하는 매력적인 언어입니다.

2. 왜 파이썬을 배워야 할까요? 파이썬의 특징과 장점

파이썬이 전 세계적으로 폭발적인 인기를 얻게 된 데에는 여러 가지 이유가 있습니다. 프로그래밍 초보자인 여러분이 파이썬을 선택해야 하는 명확한 이유들을 소개합니다.

2.1. 뛰어난 가독성과 쉬운 문법

파이썬의 가장 큰 특징이자 장점은 바로 '읽기 쉬운 코드'입니다. 다른 언어에서는 여러 줄에 걸쳐 작성해야 할 코드를 파이썬에서는 단 한두 줄로 끝내는 경우가 많습니다. 또한, 들여쓰기(Indentation)를 통해 코드 블록을 구분하는 방식은 코드의 구조를 한눈에 파악하기 쉽게 만들어줍니다. 이는 코드를 처음 배우는 사람들에게 큰 장점으로 작용하여, 복잡한 문법 규칙에 대한 부담 없이 논리적인 사고에 집중할 수 있게 돕습니다.

2.2. 무궁무진한 활용 분야

파이썬은 특정 분야에만 국한되지 않습니다. 여러분이 어떤 꿈을 꾸든, 파이썬은 그 꿈을 현실로 만드는 데 도움을 줄 수 있습니다.

  • 웹 개발: 인스타그램, 드롭박스 등 유명 서비스들이 파이썬으로 개발되었습니다. Django(장고), Flask(플라스크)와 같은 강력한 웹 프레임워크를 통해 빠르고 안정적인 웹 서비스를 만들 수 있습니다.
  • 데이터 과학 및 인공지능(AI): 파이썬은 데이터 분석, 머신러닝, 딥러닝 분야에서 독보적인 위치를 차지합니다. NumPy, Pandas, Scikit-learn, TensorFlow, PyTorch와 같은 라이브러리들은 이 분야의 사실상 표준입니다.
  • 자동화: 반복적인 업무(파일 정리, 이메일 발송, 웹 스크래핑 등)를 파이썬 스크립트 하나로 자동화하여 업무 효율을 극대화할 수 있습니다.
  • 게임 개발: Pygame(파이게임)과 같은 라이브러리를 사용해 간단한 2D 게임을 만들 수 있습니다.
  • 데스크톱 애플리케이션: PyQt, Tkinter 등을 사용해 사용자 인터페이스를 갖춘 프로그램을 만들 수 있습니다.

잠깐! 파이썬, 생각보다 더 다양하게 활용됩니다. 파이썬은 인공지능이나 데이터 과학 같은 복잡한 분야에서만 사용된다고 생각하기 쉽지만, 사실은 간단한 스크립트 작성부터 웹 개발, 자동화 등 여러분의 일상 속 다양한 문제 해결에도 활용됩니다. 초보자도 충분히 실용적인 결과물을 만들 수 있는 만능 언어입니다.

2.3. 방대한 라이브러리 생태계

파이썬의 또 다른 강력한 점은 '배터리 포함(Batteries Included)'이라는 말처럼 풍부한 라이브러리 생태계입니다. 라이브러리는 특정 기능을 미리 만들어 놓은 코드 묶음이라고 생각하시면 됩니다. 예를 들어, 수학 계산을 위한 math 라이브러리, 웹 요청을 위한 requests 라이브러리 등 수많은 라이브러리가 존재합니다. 덕분에 개발자는 모든 것을 처음부터 만들 필요 없이, 이미 만들어진 라이브러리를 가져다 써서 개발 시간을 단축하고 효율성을 높일 수 있습니다.

2.4. 활발한 커뮤니티 지원

파이썬은 전 세계적으로 수많은 개발자가 사용하는 언어인 만큼, 매우 활발한 커뮤니티를 가지고 있습니다. 궁금한 점이 생기거나 문제에 부딪혔을 때, 온라인 포럼, 커뮤니티, 스택 오버플로우(Stack Overflow) 등에서 쉽게 답을 찾고 도움을 받을 수 있습니다. 이는 특히 프로그래밍 초보자에게 큰 힘이 됩니다.

3. 파이썬 개발 환경 설정하기

이제 파이썬의 매력을 충분히 느꼈으니, 여러분의 컴퓨터에 파이썬을 설치하고 코드를 작성할 수 있는 환경을 만들어봅시다. 프로그래밍을 하려면 수학이나 컴퓨터 공학 지식이 필수적이라고 생각할 수 있지만, 파이썬은 논리적 사고와 문제 해결 능력이 더 중요하며, 비전공자도 충분히 배울 수 있습니다. 개발 환경 설정 역시 생각보다 어렵지 않습니다!

3.1. 파이썬 설치 (인터프리터 준비)

파이썬 코드를 실행하려면 먼저 컴퓨터에 파이썬 인터프리터가 설치되어 있어야 합니다. 인터프리터는 1절에서 설명했듯이, 우리가 작성한 파이썬 코드를 컴퓨터가 이해하고 실행할 수 있는 형태로 번역해주는 핵심 프로그램입니다.

파이썬은 운영체제(Operating System, OS)에 따라 설치 방법이 약간 다릅니다. 여기서는 Windows와 macOS 환경에서의 설치 방법을 안내합니다.

3.1.1. Windows에 파이썬 설치하기

  1. 파이썬 공식 웹사이트 접속: 웹 브라우저를 열고 www.python.org에 접속합니다.
  2. 다운로드 페이지 이동: 상단 메뉴에서 Downloads를 클릭합니다.
  3. 설치 파일 다운로드: Download the latest version for Windows 섹션에서 최신 버전의 파이썬 설치 파일(예: Python 3.x.x)을 다운로드합니다. 일반적으로 64비트 버전을 다운로드하면 됩니다.
  4. 설치 시작: 다운로드한 .exe 파일을 실행합니다.
  5. 중요! PATH 추가: 설치 화면이 나타나면 가장 아래에 있는 Add Python 3.x to PATH 옵션을 반드시 체크해야 합니다. 이 옵션을 체크하지 않으면 명령 프롬프트(CMD)에서 파이썬 명령어를 사용할 수 없어 불편해집니다.
    • PATH(환경 변수): 운영체제가 프로그램을 찾을 때 참고하는 경로 목록입니다. PATH에 파이썬 경로를 추가하면, 파이썬 실행 파일이 있는 폴더로 이동하지 않고도 명령 프롬프트의 어느 위치에서든 python 명령어를 실행할 수 있게 되어 매우 편리합니다.
  6. 설치 진행: Install Now를 클릭하여 설치를 시작합니다.
  7. 설치 완료: 설치가 완료되면 Setup was successful 메시지가 나타납니다. Close를 클릭하여 창을 닫습니다.

3.1.2. macOS에 파이썬 설치하기

macOS에는 기본적으로 파이썬 2.x 버전이 설치되어 있을 수 있지만, 현재 대부분의 파이썬 개발은 파이썬 3.x 버전으로 이루어지므로 최신 버전을 새로 설치하는 것이 좋습니다.

  1. 파이썬 공식 웹사이트 접속: 웹 브라우저를 열고 www.python.org에 접속합니다.
  2. 다운로드 페이지 이동: 상단 메뉴에서 Downloads를 클릭합니다.
  3. 설치 파일 다운로드: Download the latest version for macOS 섹션에서 최신 버전의 파이썬 설치 파일(예: Python 3.x.x macOS 64-bit installer)을 다운로드합니다.
  4. 설치 시작: 다운로드한 .pkg 파일을 실행합니다.
  5. 설치 진행: 설치 프로그램의 지시에 따라 계속, 동의, 설치 버튼을 클릭하여 설치를 진행합니다. 특별한 설정을 변경할 필요는 없습니다.
  6. 설치 완료: 설치가 완료되면 요약 화면이 나타납니다. 닫기를 클릭하여 창을 닫습니다.

3.1.3. 파이썬 설치 확인

설치가 제대로 되었는지 확인하는 가장 좋은 방법은 터미널(Windows: 명령 프롬프트 또는 PowerShell, macOS: 터미널)을 여는 것입니다.

  1. 터미널 열기:
    • Windows: Windows 키 + R을 누르고 cmd를 입력한 후 Enter를 누릅니다. 또는 시작 메뉴에서 명령 프롬프트를 검색하여 실행합니다.
    • macOS: Command + Space를 누르고 터미널을 입력한 후 Enter를 누릅니다.

  2. 버전 확인: 터미널 창에 다음 명령어를 입력하고 Enter를 누릅니다. bash python --version 또는 bash python3 --version 정상적으로 설치되었다면 Python 3.x.x와 같은 형태로 파이썬 버전 정보가 출력될 것입니다. 만약 python 명령어로 버전이 나오지 않는다면 python3로 시도해 보세요. (macOS에서는 기본적으로 구버전 파이썬 2가 설치되어 있을 수 있으므로, 새로 설치한 파이썬 3 버전을 명확히 사용하기 위해 python3 명령어를 사용하는 것이 좋습니다.)

    ```

    예시 출력

    Python 3.10.6 ```

3.2. 통합 개발 환경(IDE) 설정: VS Code

파이썬 코드를 작성하고 실행하는 데에는 '메모장' 같은 일반 텍스트 편집기도 사용할 수 있습니다. 하지만 훨씬 더 효율적이고 편리하게 개발할 수 있도록 도와주는 도구가 있습니다. 바로 통합 개발 환경(Integrated Development Environment, IDE)입니다. IDE는 코드 편집, 디버깅(오류 찾기), 실행 등 개발에 필요한 모든 기능을 한곳에 모아 제공하는 소프트웨어입니다.

여기서는 가장 인기 있고 강력하며 무료인 Visual Studio Code (VS Code)를 사용하여 개발 환경을 설정하는 방법을 소개합니다.

3.2.1. VS Code 설치하기

  1. VS Code 공식 웹사이트 접속: 웹 브라우저를 열고 code.visualstudio.com에 접속합니다.
  2. 다운로드: 운영체제(Windows 또는 macOS)에 맞는 Download 버튼을 클릭하여 설치 파일을 다운로드합니다.
  3. 설치 시작: 다운로드한 설치 파일을 실행합니다.
  4. 설치 진행:
    • Windows: 동의함에 체크하고 다음 버튼을 계속 클릭하여 설치를 완료합니다. 중간에 PATH에 추가 옵션이 있다면 체크하는 것이 좋습니다.
    • macOS: 다운로드한 .zip 파일을 더블클릭하여 압축을 해제하면 Visual Studio Code.app 파일이 나타납니다. 이 파일을 Finder의 응용 프로그램 폴더(왼쪽 사이드바에 위치)로 드래그하여 옮기면 설치가 완료됩니다.
  5. VS Code 실행: 설치가 완료되면 VS Code를 실행합니다.

3.2.2. VS Code에서 파이썬 확장(Extension) 설치하기

VS Code는 다양한 프로그래밍 언어를 지원하지만, 파이썬 개발에 특화된 기능을 사용하려면 '파이썬 확장'을 설치해야 합니다. 이 확장은 코드 자동 완성, 문법 오류 검사, 디버깅, 코드 실행 등 파이썬 개발에 필수적인 기능을 제공합니다.

  1. 확장 탭 열기: VS Code 왼쪽 사이드바에서 네모 상자 4개가 겹쳐진 아이콘(Extensions)을 클릭합니다. 또는 Ctrl+Shift+X (Windows) 또는 Cmd+Shift+X (macOS)를 누릅니다.
  2. 파이썬 확장 검색: 검색창에 Python을 입력합니다.
  3. 설치: Microsoft에서 제공하는 Python 확장을 찾아 Install 버튼을 클릭합니다. (가장 위에 있으며, 다운로드 수가 압도적으로 많은 확장입니다.)
  4. 설치 완료 확인: Install 버튼이 Uninstall로 바뀌면 설치가 완료된 것입니다.

3.3. 인터프리터와 IDE의 상호작용

이제 여러분의 컴퓨터에는 파이썬 코드를 실제로 실행하는 인터프리터와 코드를 편리하게 작성하고 관리하는 VS Code (IDE)가 모두 준비되었습니다. 이 둘은 서로 다른 역할을 하지만, 파이썬 개발을 위해서는 긴밀하게 협력해야 합니다.

VS Code는 그 자체로 파이썬 코드를 실행할 수 있는 능력을 가지고 있지 않습니다. VS Code는 여러분이 코드를 더 쉽게 작성하고, 오류를 찾고, 프로젝트를 관리할 수 있도록 도와주는 '편리한 작업 환경'을 제공할 뿐입니다. 여러분이 VS Code에서 파이썬 코드를 실행하라는 명령을 내리면, VS Code는 내부적으로 설치된 파이썬 인터프리터에게 해당 코드를 전달하고 실행을 요청합니다. 인터프리터가 코드를 번역하고 실행한 후, 그 결과를 다시 VS Code에 보여주는 방식인 것이죠.

VS Code의 파이썬 확장은 이러한 상호작용을 매우 매끄럽게 만들어줍니다. 확장 덕분에 VS Code는 여러분의 시스템에 설치된 여러 파이썬 인터프리터(예: 파이썬 3.9, 파이썬 3.10 등)를 자동으로 감지하고, 어떤 인터프리터를 사용할지 쉽게 선택할 수 있는 기능을 제공합니다. 이를 통해 여러분은 복잡한 설정 없이도 원하는 파이썬 버전으로 코드를 실행할 수 있습니다.

이제 여러분의 컴퓨터는 파이썬 코드를 작성하고 실행할 준비를 마쳤습니다. 개발 환경 설정이 매우 어렵고 복잡할 것이라고 지레짐작할 수 있지만, 보시다시피 공식 설치 프로그램과 VS Code 같은 도구를 활용하면 비교적 쉽고 빠르게 환경을 구축할 수 있습니다.

4. 나의 첫 파이썬 프로그램: 'Hello, World!'

프로그래밍 세계에 오신 것을 환영합니다! 이제 모든 준비가 끝났으니, 프로그래밍 학습의 오랜 전통인 "Hello, World!" 프로그램을 만들어 봅시다. 이 간단한 코드는 여러분이 작성한 코드가 실제로 작동한다는 것을 보여주는 첫 번째 증거가 될 것입니다.

4.1. 코드 작성하기

  1. VS Code 열기: VS Code를 실행합니다.
  2. 새 파일 생성:
    • File (파일) > New File (새 파일)을 클릭합니다.
    • File (파일) > Save As... (다른 이름으로 저장...)를 클릭합니다.
    • 파일 이름을 hello.py로 지정하고, 저장할 위치(예: 바탕화면이나 문서 폴더 안에 python_projects 폴더를 만들어서)를 선택한 후 저장을 클릭합니다. .py는 파이썬 파일의 확장자입니다.
  3. 코드 입력: hello.py 파일에 다음 코드를 입력합니다. python print("Hello, World!") 정확히 입력했는지 확인하고 파일을 저장합니다 (Ctrl+S 또는 Cmd+S).

4.2. 코드 실행하기

VS Code에서 파이썬 코드를 실행하는 방법은 여러 가지가 있습니다. 가장 일반적인 두 가지 방법을 소개합니다.

4.2.1. VS Code의 터미널에서 실행하기

  1. 터미널 열기: VS Code 상단 메뉴에서 Terminal (터미널) > New Terminal (새 터미널)을 클릭합니다. VS Code 하단에 터미널 창이 열립니다.
  2. 코드 실행: 터미널 창에 다음 명령어를 입력하고 Enter를 누릅니다. bash python hello.py 또는 macOS에서는 bash python3 hello.py 터미널에 Hello, World!라는 메시지가 출력되는 것을 볼 수 있을 것입니다. 축하합니다! 여러분의 첫 번째 파이썬 코드가 성공적으로 실행되었습니다.

4.2.2. 파이썬 인터프리터(REPL)에서 직접 실행하기

파이썬은 REPL (Read-Eval-Print Loop) 환경을 제공합니다. 이는 코드를 한 줄씩 입력하고 즉시 결과를 확인할 수 있는 대화형 모드입니다. 간단한 코드를 테스트하거나 파이썬 문법을 탐색할 때 유용합니다.

  1. 터미널 열기: 위에서 설명한 대로 명령 프롬프트(Windows) 또는 터미널(macOS)을 엽니다.
  2. 파이썬 인터프리터 시작: 터미널에 python 또는 python3를 입력하고 Enter를 누릅니다. bash python # 또는 python3 프롬프트가 >>>로 바뀌면 파이썬 인터프리터가 시작된 것입니다.
  3. 코드 입력 및 실행: >>> 뒤에 다음 코드를 입력하고 Enter를 누릅니다. python >>> print("Hello, World!") 입력과 동시에 Hello, World!이라는 결과가 출력될 것입니다.
  4. 인터프리터 종료: exit()를 입력하거나 Ctrl+Z (Windows) 또는 Ctrl+D (macOS)를 누르고 Enter를 누르면 인터프리터 모드에서 나옵니다.

5. 코드 가독성을 높이는 주석(Comment)

코드를 작성하다 보면 단순히 실행되는 것 이상으로 '이 코드가 무엇을 하는지' 명확하게 전달하는 것이 중요합니다. 특히 다른 사람과 협업하거나 나중에 자신이 작성한 코드를 다시 볼 때, 코드가 어떤 목적으로 작성되었는지 파악하기 어려울 수 있습니다. 이때 주석(Comment)이 큰 역할을 합니다.

주석은 코드에 대한 설명을 작성하는 부분으로, 파이썬 인터프리터가 코드를 실행할 때 주석 부분은 완전히 무시합니다. 즉, 주석은 프로그램의 동작에 아무런 영향을 주지 않습니다.

5.1. 한 줄 주석: #

파이썬에서 한 줄 주석은 # 기호로 시작합니다. # 뒤에 오는 모든 내용은 해당 줄의 끝까지 주석으로 처리됩니다.

# 이것은 한 줄 주석입니다.
# 이 코드는 화면에 "Hello, World!"를 출력합니다.
print("Hello, World!") # print() 함수는 괄호 안의 내용을 출력합니다.

주석은 코드의 맨 앞에 올 수도 있고, 코드의 특정 부분에 대한 설명을 위해 줄 중간에 올 수도 있습니다.

5.2. 여러 줄 주석 (Docstring): """ 또는 '''

여러 줄에 걸쳐 긴 설명을 추가해야 할 때는 세 개의 큰따옴표(""")나 작은따옴표(''')로 감싸는 방식을 사용합니다. 이는 주로 함수나 클래스, 모듈(파일) 전체에 대한 설명을 작성할 때 사용되며, 독스트링(Docstring)이라고 부릅니다. 독스트링은 단순히 주석 기능을 넘어, 코드의 문서화에도 활용됩니다.

"""
이것은 여러 줄 주석입니다.
이 파일은 파이썬 기초 학습을 위한 첫 번째 예제 코드입니다.
'Hello, World!'를 출력하여 파이썬 설치 및 실행을 확인합니다.
"""

# print() 함수를 사용하여 문자열을 화면에 출력합니다.
print("Hello, World!")

주석을 적절히 활용하면 코드의 가독성이 크게 향상되고, 나중에 코드를 수정하거나 다른 사람과 공유할 때 큰 도움이 됩니다. 좋은 개발자는 항상 깔끔하고 이해하기 쉬운 코드를 작성하기 위해 노력하며, 주석은 그 노력의 중요한 부분입니다.

6. 기본 출력 함수: print()

우리가 방금 사용한 print() 함수는 파이썬에서 가장 기본적이면서도 중요한 함수 중 하나입니다. 함수는 특정 작업을 수행하도록 미리 정의된 코드 블록이며, print() 함수는 괄호 안에 있는 내용을 화면에 출력하는 역할을 합니다.

print() 함수를 사용하여 다양한 형태의 데이터를 출력하는 방법을 알아봅시다.

6.1. 문자열 출력

문자열(String)은 글자들의 나열을 의미하며, 파이썬에서는 큰따옴표(")나 작은따옴표(')로 감싸서 표현합니다.

print("안녕하세요, 파이썬!")
print('Python is fun!')

결과:

안녕하세요, 파이썬!
Python is fun!

6.2. 숫자 및 연산 결과 출력

print() 함수는 숫자나 간단한 연산의 결과도 출력할 수 있습니다.

print(123)       # 정수 출력
print(3.14)      # 실수 출력
print(10 + 5)    # 덧셈 연산 결과 출력
print(20 * 3)    # 곱셈 연산 결과 출력

결과:

123
3.14
15
60

6.3. 여러 값 한꺼번에 출력

print() 함수는 쉼표(,)를 사용하여 여러 개의 값을 한꺼번에 출력할 수 있습니다. 이때 각 값은 기본적으로 공백으로 구분되어 출력됩니다.

print("이름:", "홍길동", "나이:", 30)
print("파이썬은", "정말", "재미있어요!")

결과:

이름: 홍길동 나이: 30
파이썬은 정말 재미있어요!

6.4. 줄 바꿈 없이 출력하기 (선택 사항)

print() 함수는 기본적으로 출력 후 자동으로 줄 바꿈을 합니다. 하지만 end 매개변수를 사용하여 줄 바꿈 동작을 변경할 수 있습니다. 예를 들어, end=""를 사용하면 줄 바꿈 없이 다음 출력이 바로 이어집니다.

print("Hello", end=" ")
print("World!")

결과:

Hello World!

또한, sep 매개변수를 사용하여 여러 값을 출력할 때 기본 공백 대신 다른 구분자를 사용할 수도 있습니다.

print("사과", "바나나", "딸기", sep=", ")

결과:

사과, 바나나, 딸기

end와 sep 매개변수는 당장 필수는 아니지만, print() 함수의 유연성을 보여주는 좋은 예시입니다. 나중에 필요할 때 다시 찾아보세요.
